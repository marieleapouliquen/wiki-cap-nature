# Cap Nature — Ateliers Birds

Guide libre pour animer les **ateliers Cap Nature — Birds** : (re)découvrir la nature avec des capteurs participatifs.

> À destination des enseignant·e·s et éducateur·ice·s nature.

## Présentation

Les ateliers Cap Nature — Birds sont des dispositifs pédagogiques d'environ 75 minutes qui combinent l'usage de capteurs numériques (parabole acoustique, application d'identification des chants d'oiseaux, sonomètre) et l'exploration sensible d'un site naturel ou urbain.

## Contenu du site

- **[Accueil](docs/index.html)** — présentation et aperçu de l'atelier
- **[Protocole](docs/protocole/index.html)** — déroulé complet en 3 phases, timing, posture d'animation
- **[Matériel](docs/materiel/index.html)** — Merlin Bird ID, parabole directionnelle, FizziQ, budgets et alternatives
- **[Ressources](docs/ressources/index.html)** — sélection de sites, applications, associations et lectures externes

## Déployer sur GitHub Pages

1. **Settings → Pages**
2. Source : branche `main`, dossier `/docs`
3. Le site sera accessible à `https://<utilisateur>.github.io/<nom-du-depot>/`

C'est du HTML statique pur, sans compilation. Il suffit d'ouvrir `docs/index.html` localement pour le consulter.
